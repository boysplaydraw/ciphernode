Özellikler

Gizlilik, uzlaşı değil bir standart

Her özellik sıfırdan gizlilik göz önünde bulundurularak tasarlandı.

🔐
OpenPGP RSA-4096
Mesajlar gönderilmeden önce alıcının açık anahtarıyla şifrelenir. Sunucu şifreli veriyi asla göremez. İmzalama desteğiyle kimlik doğrulama.
🧅
Tor Ağı Desteği
Masaüstünde Tor otomatik yönetilir. Mobilde Orbot ile tüm trafik Tor'a yönlenir. .onion adresli kendi relay sunucunuzu kurabilirsiniz.
⚡
P2P Mesajlaşma
Sadece P2P modunda relay sunucusu devre dışı kalır. Mesajlar sadece alıcı çevrimiçiyken iletilir. Çevrimdışı kuyruk opsiyoneldir.
📁
Şifreli Dosya Paylaşımı
Dosyalar client-side şifrelendikten sonra yüklenir. İndirme limiti ve 24 saatlik TTL. AES anahtarı alıcının PGP anahtarıyla sarmalanır.
🎲
Anonim Eşleştirme
Gerçek kimlik paylaşılmadan rastgele kullanıcılarla bağlantı kurulur. Takma ad sistemi ile oturumlar izole edilir.
🔤
Steganografi Modu
PGP şifreli payload'ı görünmez Unicode karakterlere gömer. Normal metin gibi görünen mesajlar aslında şifreli veri taşır.
👻
Hayalet Modu
Yazıyor göstergesi ve okundu bilgisi gönderilmez. Karşı taraf ne zaman yazdığınızı bilemez.
🌐
Çapraz Platform
Android, iOS, Windows, macOS, Linux ve Web tarayıcısında aynı deneyim. Electron masaüstü, React Native mobil, Expo web.
🏠
Kendi Sunucun
Docker, Termux, CasaOS, Unraid ile dakikalar içinde kendi relay sunucunuzu kurun. Kayıt yok, izleme yok, tam kontrol.

Nasıl Çalışır

Sade ve güvenli

01

Kimlik Oluştur

Uygulama açıldığında RSA-4096 anahtar çifti otomatik oluşturulur. Hiçbir kişisel bilgi istenmez.

02

Kişi Ekle

8 karakterli kimlik kodu veya QR ile kişi ekleyin. Açık anahtar exchange otomatik gerçekleşir.

03

Şifreli Mesajlaş

Her mesaj gönderilmeden önce alıcının açık anahtarıyla şifrelenir. Relay sunucusu şifreli veriyi göremez.

04

Tor ile Anonimleş

Tor'u etkinleştirince trafik Tor ağı üzerinden geçer. IP adresiniz relay sunucusundan da gizlenir.

Kendi Sunucu

Tam kontrol sizde

Relay sunucusunu kendi altyapınızda çalıştırın. Herkes sizin sunucunuza bağlanabilir, ya da kendi sunucusunu kurabilir.

🐳
Docker
Linux · macOS · Windows
# Repo'yu klonla
git clone https://github.com/boysplaydraw/ciphernode
cd ciphernode

# Başlat
docker compose up -d
Port 5000 otomatik açılır. SSL için Cloudflare Tunnel veya scripts/setup-ssl.sh kullanın.
📱
Termux (Android)
Android telefon = sunucu
# Termux'ta çalıştır
bash termux-start.sh

# Tor hidden service ile
bash termux-start.sh tor

# Cloudflare Tunnel ile
bash termux-start.sh cloudflare
Termux F-Droid'den indirilebilir. Tüm bağımlılıklar otomatik kurulur.
🖥
Linux / VPS
Ubuntu · Debian · Arch
# Sunucu başlat
npm run server:build
npm run server:prod

# SSL (Let's Encrypt + nginx)
sudo bash scripts/setup-ssl.sh \
  relay.example.com
Otomatik systemd servisi, sertifika yenileme ve reverse proxy kurulur.
🌐
ngrok / Cloudflare Tunnel
CGNAT arkasında bile çalışır
# ngrok (SSL otomatik)
bash scripts/setup-ngrok.sh TOKEN

# Cloudflare (ücretsiz, kayıt yok)
bash scripts/setup-cloudflare.sh
URL otomatik gösterilir. Uygulamada Özel Sunucu olarak girin.
🏠
CasaOS / Umbrel
NAS · Raspberry Pi · Home Server
# CasaOS App Store'dan
Custom App → Docker Compose:

image: ciphernode/relay
ports: "5000:5000"
casaos/ ve unraid/ klasörlerinde hazır manifest dosyaları bulunur.
🧅
Tor Hidden Service
.onion — tamamen anonim
# .onion adresi oluştur
bash scripts/setup-tor-hidden-service.sh

# Çıktı:
.onion: abc123...xyz.onion
Port forwarding gerekmez. Sunucu IP'si tamamen gizli kalır.

SSS

Sık Sorulan Sorular

Merkezi bir sunucu var mı? ▼
CipherNode'un herhangi bir resmi merkezi sunucusu yoktur. Kullanıcılar kendi relay sunucularını kurar veya başkasının sunucusuna bağlanır. Tüm relay sunucuları mesajları RAM'de tutar ve iletimden sonra siler. Kayıt dosyası oluşturulmaz.
Relay sunucusu mesajları okuyabilir mi? ▼
Hayır. Mesajlar gönderilmeden önce alıcının açık anahtarıyla şifrelenir (OpenPGP RSA-4096). Relay sunucusu sadece şifreli veriyi iletir ve asla içeriğe erişemez. İmzalı mesajlar için gönderici kimliği de doğrulanabilir.
Tor Browser web uygulamasında çalışır mı? ▼
Evet. Web uygulamasını Tor Browser'da açarsanız tüm trafik otomatik olarak Tor üzerinden geçer. Ekstra ayar gerekmez. Relay sunucunuz .onion adresiyle çalışıyorsa Ağ Ayarları'ndan özel sunucu olarak girebilirsiniz.
CGNAT arkasındayım, sunucu kurabilir miyim? ▼
Evet. Cloudflare Tunnel veya ngrok kullanarak CGNAT arkasından da sunucuyu dışarıya açabilirsiniz. Bu yöntemler port forwarding gerektirmez. Tor hidden service (.onion) da CGNAT'ı tamamen aşar.
Kimliğimi birden fazla cihazda kullanabilir miyim? ▼
Evet. Ayarlar → Kimliği Dışa Aktar ile JSON backup alabilirsiniz. Başka cihazda Kimliği İçe Aktar ile aynı PGP anahtarlarını ve kimlik kodunuzu yükleyebilirsiniz.
Açık kaynak mı? ▼
Evet. Tüm kaynak kodu GitHub'da açık olarak yayınlanmaktadır. Bağımsız güvenlik araştırmacıları tarafından incelenebilir.
